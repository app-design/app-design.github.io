Home Page

移动开发框架提供了模板首页，因为每个项目的特殊性，此模板首页也只能当参考，不具有普遍性。首页一般都是按钮组合、列表、轮播图、卡片等组合而成。

设计首页要有主次之分#

突出重点，色彩不宜过多，操作不宜过于复杂。

间距关系要合理#

不要太密，各个组件不宜太紧凑。为了适用不同尺寸的屏幕，在横向采用合理的布局来排布组件，从而保证布局的灵活性。
