Git Guide

介绍#

这是高伟达移动团队的 Git 分支管理规范。每个人对工具的使用往往各有偏好，各种方法各有利弊，无所谓对错。但涉及团队协作的方面需要有一些一致的规范，所以请大家务必遵守。

除了一致性之外，这个规范有以下几个目的：

  • 建立产品开源协作流程和规范；

  • 帮助产品和社区用户取得共赢；

  • 为高伟达的开源事业作出有益的尝试。

分支管理规范#

app-design 托管在高伟达内部 Git 仓库 Gogs 上。产品团队和项目实施团队，以 Gogs 平台为依托，根据协同模式的不同，分别使用以下不同的分支管理规范最佳实践：

  • 产品团队：产品协作规范

  • 实施团队：开源协作规范
